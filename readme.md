## Installation Instruction

```
  $ > composer install
```

## Clear previous laravel config
```
  $ > php artisan config:clear
```
## or

```
  $ > cache:clear
  $ > config:clear
  $ > route:clear
  $ > session:table
  $ > view:clear
```
